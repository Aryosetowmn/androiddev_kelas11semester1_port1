# Portfolio Application

A creative and feature-rich Android application designed to showcase a personal portfolio, curriculum vitae (CV), and projects. This project demonstrates advanced Android development practices, including RecyclerView, custom adapters, and side navigation menus.

## Introduction

The Portfolio Application is a multi-functional app designed to highlight a portfolio of projects, work experience, and educational background. It provides an elegant user experience and serves as a tool for showcasing professional achievements.

---

## Features
- Display a user-friendly portfolio gallery with project images.
- Curriculum Vitae (CV) section with detailed education and experience history.
- Side navigation menu with customizable options.
- Modular architecture with reusable components.
- Clean and responsive UI with RecyclerView integration.

---

## Technologies Used
- Java
- Android Studio
- RecyclerView and CardView for lists
- Picasso library for image loading

---

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Aryosetowmn/androiddev_kelas11semester1_port1.git
   ```
2. **Open the project** in Android Studio.
3. Sync and configure dependencies using Gradle.
4. **Run the app** on an emulator or physical Android device.

---

## Usage

- Navigate through the side menu to explore the portfolio, CV, and featured sections.
- Interact with clickable portfolio items to learn more about each project.
- Use the CV section to view educational and professional milestones.
